Properties of determinant

  1. The value of a determinant remains unaltered by changing its rows into columns and columns into rows
  2. If the two rows or columns of a determinant are interchanged, then the determinant changes by sign.
  3. If two rows or columns of a determinant are equal, then its determinant is equal to zero.
  4. If each element of a row or a column of a determinant is multiplied by scalar k, then the value of the determinant is also multiplied by k.
  5. If a row (or column) or a multiple of any row (or column) is added or subtracted from any other row (or column), then the value of the determinants remains unchanged.
